# Task Management System

A full-stack Task Management System built with Node.js + TypeScript (Backend) and Next.js + TypeScript + TailwindCSS (Frontend).  
Features : JWT Authentication, CRUD tasks, search & filter, toggle status, and a responsive UI.

---

## 🔹 Features

### Authentication
- Register, Login, Logout  
- JWT Authentication with Access Token (short-lived) and Refresh Token (long-lived)  
- Password hashing using **bcrypt**  

### Task Management
- Create, View, Update, Delete tasks  
- Toggle task status (Done / Pending)  
- Pagination, search by title, filter by status  

### Frontend
- Login & Registration pages  
- Task Dashboard (CRUD + search + filter + toggle)  
- Toast notifications for actions  
- Responsive design (desktop + mobile)
